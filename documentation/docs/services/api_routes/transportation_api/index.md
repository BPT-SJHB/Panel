# TransportationAPI

### شماره پورت: ۸۲

---

این API مربوط به فرایند های حمل و نقلی سیستم مانند:

- خودرو
- راننده
- شرکت حمل‌ونقل
- کارخانجات
- مخازن بار
- شهرها و استان ها
- انواع بارگیرها
- انواع کالا
- طول سفر
- تعرفه ها
- صفوف نوبت‌دهی
- اعلام بارها
- نوبت‌ها و مجوز های بارگیری

است که هرکدام با مسیرهای خاص مشخص شده اند.
